myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies!

Method Request Request data type URL Response Example
GET Get a list of all movies X /movies A JSON object containing data on all movies
GET Get data about a single movie by title X /movies/:Title A JSON object containing data on a single movie title
GET Get data about a genre by name X /movies/genre/:Name A JSON object containing data on a single genre
GET Get data about a director by name X /movies/director/:Name A JSON object containing data on a single director
POST New user registration JSON object including at least a name /users A text message indicating a successful/unsuccessful registration
POST Add a movie to your list of favorites by title X /users/:Username/movies/:MovieID A text message indicating a movie was successfully/unsuccessfully added to favorites
PUT Update user information X /users/username/:username A text message indicating user information was successfully/unsuccessfully updated
DELETE Remove a movie from your list of favorites X /users/:Username/movies/:MovieID A text message indicating a movie was successfully/unsuccessfully removed from favorites
DELETE Remove user registration X /users/:Username A text message indicating the user was successfully/unsuccessfully removed
