myFlix

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies!

Method Request Request data type URL Response Example
GET Get a list of all movies X /movies A JSON object containing data on all movies
GET Get data about a single movie by title X /movies/:Title A JSON object containing data on a single movie title
GET Get data about a genre by name X /movies/genre/:Name A JSON object containing data on a single genre
GET Get data about a director by name X /movies/director/:Name A JSON object containing data on a single director
GET Get data about a single user by Username X /users/:Username A JSON object containing data on a single director
POST New user registration A JSON object including at least a key-value pair for Username /users A welcome message indicating a successful registration
POST Add a movie to your list of favorites by title X /users/:Username/movies/:MovieID A message indicating a movie was successfully added to favorites
PUT Update user information A JSON object containing at least one key-value pair /users/:username A message indicating user information was successfully updated
DELETE Remove a movie from your list of favorites X /users/:Username/movies/:MovieID A message indicating a movie was successfully removed from favorites
DELETE Remove user registration X /users/:Username A message indicating the user was successfully removed
